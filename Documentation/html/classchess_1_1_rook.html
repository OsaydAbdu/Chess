Position [] chess.Rook.getPossibleMoves ( )

This method finds all the possible moves of the current piece based on the its location and the movement style of the current piece. Notice: the possible moves are not checked for validity (they could also be out of bounds), so the result should be filtered by the isValidMove() method.

◆ isValidMove()

boolean chess.Rook.isValidMove ( Cell  targetCell,
boolean  occupiedMatrix[][] 
)

This method evaluates whether the target cell's position would be a valid move for the current piece based on these conditions: 1- The movement style is a valid movement style (vertically or horizontally). 2- There is no intermediate pieces between the current position and the target cell. This method assumes that it is called after checking that the target cell is valid.

Parameters
