Detailed Description

This class is written for the chess package during the CS242 class and it is copyrighted for the author.

Author
Osayd Abdu @netid abdu2
Version
1.1 This class describes the Controller of the MVC, it is responsible for receiving the user actions from the GameView class and send the appropriate action to the ChessGame class, as it waits in the game loop for any action. It is the only way for the gameView and ChessGame classes to interact with each other.
