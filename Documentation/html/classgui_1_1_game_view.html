void gui.GameView.undo ( Move  prevMove)

This method is responsible for the undo command in the view part. It takes the argument prevMove and swaps the source and target using the Move class method swap(). Then, it transfer the icon from the source (the new source after the swap) to the target, and sets the source icon according the deletedPiece object under the prevMove object.
